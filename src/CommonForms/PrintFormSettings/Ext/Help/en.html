Spreadsheet Document Page Setup

To configure the page setup of a spreadsheet document, select Page setup.

In the window that opens, you can specify the paper size and orientation, define margins, select headers and footers, enter the scale and the number of pages per sheet to be printed.

Paper
The Size field allows you to select the required paper size.
Orientation
Use the radio button to select between Portrait and Landscape print orientations.
Margins
Use the Left, Right, Top, and Bottom fields to specify the margins for printing.
Headers and footers
The ‘Headers and footers’ group is intended to specify the size of the sheet area that will be used to print headers and footers. Sizes are entered in millimeters and calculated from the top or bottom margin to the respective sheet edge.
Scaling
Using the Adjust to and Adjust to Fit radio buttons you can specify the scale to be used to print the spreadsheet. 
The Adjust to field is intended to enter a scale to apply when printing the spreadsheet (percentage). This scale does not affect the document viewing scale in the spreadsheet editor.
If Adjust to Fit is checked, the scale is calculated automatically so that the document fits one sheet (by width) during printing.
Pages per sheet 
Allows you to select the number of document pages that will be located on a single sheet upon printing (pages are located one under another).
If the Auto option is checked, the number of pages will be determined automatically: if the selected sheet size and margins allow you to fit 2 document pages in full, two pages will be printed (otherwise only one page).
Print in 
If Black & White mode is enabled, the colors specified for the spreadsheet will be ignored and the document will be printed in black and white.
